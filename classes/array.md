# 数组

## size
::: tip 
返回值类型：int
:::


- 获取数组的长度
```javascript
${arrayVar.size()}
```
## join
| 参数名 |       描述       | 可否为空 |
| :----: | :--------------: | :------: |
| separator |  分割字符  | 是 |
::: tip 
返回值类型：String
:::

- 将数组用separator拼接在一起
```javascript
${arrayVar.join(',')}
```

## toList
::: tip 
返回值类型：List
:::


- 将数组转换成List
```javascript
${arrayVar.toList()}
```
